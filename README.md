# Elderly-transportation

## Project Introduction: Why We Built This App
In today's rapidly urbanizing environment, public transportation networks are becoming increasingly complex. For the elderly, navigating these systems to access accurate information can be a daunting task. Many existing transportation apps are cluttered with unnecessary data and complicated interfaces, creating significant cognitive burden for senior users.

The "Elderly-transportation" project was created with a **Minimalist Design** philosophy. We aim to lower the digital barrier, providing a direct and intuitive way for the elderly to access essential bus routes and transit information.

### Special Concession Schemes
This app is designed to provide clear information regarding the following public transport fare schemes:
*   **$2 Concessionary Fare Scheme**: A special rate designed to encourage social participation among the elderly.
*   **80% Off Discount**: Detailed guidance on applicable routes and conditions for the 80% discount scheme, ensuring seniors can maximize their travel benefits with ease.

## How It Works: Core Concepts
This project utilizes a local web architecture that prioritizes "lightweight, speed, and usability."

*   **Data-Driven**: The system dynamically fetches and parses CSV-based transit databases via client-side JavaScript, ensuring that the application remains portable and easy to manage locally.
*   **Parsing Mechanism**: We utilize [PapaParse](https://www.papaparse.com/), a high-performance CSV parser, to instantly transform raw data into a user-friendly format within the browser.
*   **Logic Architecture**:
    1.  User enters a search query in the interface.
    2.  The system reads the local `data.csv` file.
    3.  JavaScript processes the data to filter and match relevant results.
    4.  Results are rendered instantly on the page without requiring external server dependencies.

## Technology Stack
*   **Frontend**: HTML5, CSS3, JavaScript (ES6+)
*   **Data Processing**: [PapaParse](https://www.papaparse.com/)

## How to Compile & Run
This project is a standalone web application and requires no complex compilation steps.

### Setup Instructions
1.  **Clone or Download**: Save this repository to your local machine.
2.  **Ensure File Integrity**: Make sure `index.html`, `app.js`, `styles.css`, and `data.csv` are in the same folder.
3.  **Running the App**: Due to modern browser security policies regarding local file access, simply double-clicking `index.html` might not load the CSV data.
    *   **Recommended Method**: Use the **"Live Server"** extension in VS Code. Simply right-click `index.html` and select "Open with Live Server."
    *   **Terminal Method**: If you have Python installed, navigate to the project directory in your terminal and run:
```bash
        python -m http.server 8000
        ```
        Then, open your browser and visit `http://localhost:8000`.
